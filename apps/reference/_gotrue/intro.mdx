The Supabase Auth Server (GoTrue) is a JSON Web Token (JWT)-based API for managing users and issuing access tokens.

GoTrue is an open-source API written in Golang, that acts as a self-standing API service for handling user registration and authentication for JAMstack projects. It's based on OAuth2 and JWT and handles user signup, authentication, and custom user data.

## Docker images

Pull the latest version of the Supabase GoTrue library:
```
docker pull supabase/gotrue:latest
```

Run the image in a Docker container:
```
docker run -it gotrue
```

You can also view older versions on [Docker Hub](https://hub.docker.com/repository/docker/supabase/gotrue).
